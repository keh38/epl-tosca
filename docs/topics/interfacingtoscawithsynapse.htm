This document describes how to coordinate the measurement of behavioral data using Tosca with the measurement of physiological data using the TDT Synapse software. The diagram below illustrates the components of the Tosca-Synapse system. There are two physical connections between the systems that must be set up correctly:


 

1. An Ethernet connection between the PXI running Tosca and the PC running the TDT software. Ideally, this is a direct, dedicated connection between the two computers, to minimize communication latency

2. A cable carrying digital sync pulses from the PXI hardware to the TDT hardware.


 


 

This version of the documentation is accurate as of Tosca revision 3329.
