Running

 

 

 

Be sure to start the TDT Synapse Interface on the PC connected to the TDT hardware and running Synapse.


 

It should be started after starting Synapse, or else it will quit with an error. (That error is misleadingly labeled as an "out-of-memory" error, but it simply indicates that the TDT Synapse Interface was unable to establish a link to the Synapse software.)
