Add trial counter

Interfacing Tosca with Synapse ›› Setting up Synapse ›› Set up Experiment ››

  

 

Add a trial counter to the Synapse experiment. The number of trials acquired by Synapse will be reported to Tosca through the Synapse Interface. Tosca uses this information primarily to know when Synapse has written all of the data to disk and it is safe to end the run.


 

Under "Gizmos" in Synapse, select Custom > TrialCounter and drag onto the TDT processor. This is a custom Gizmo that should have been installed as part of the Synapse Interface installation.
