Setting up Synapse

 

 

 

There are several steps required to configure Synapse to interact correctly with Tosca. These steps are detailed in the following sections. Most of these should only need to be done one time. But in the event things stop working as expected, a useful troubleshooting step is to review these settings to make sure they are still set correctly.
